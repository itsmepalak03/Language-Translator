# Language-Translator

It is a modern, responsive, and user-friendly language translation web application that enables users to translate text between multiple languages instantly. Built using HTML, CSS, and JavaScript, it provides a clean interface, language detection, speech support, translation history, dark mode, and real-time translation using free translation APIs.

---
Features
🌍 Multi-Language Translation
- Translate text between 60+ languages
- Auto language detection
- Fast and accurate translations

🎙️ Text-to-Speech
- Listen to translated text
- Supports multiple language voices
- Play and stop speech anytime

📋 Copy & Share
- Copy source or translated text
- Share translations directly
- One-click clipboard support

🔄 Language Swap
- Instantly swap source and target languages
- Automatically updates translation panel

📚 Translation History
- Stores previous translations
- Restore previous translations instantly
- Clear history whenever needed

🌙 Dark & Light Theme
- Modern dark mode support
- Theme preference saved locally

📱 Fully Responsive
- Mobile-friendly design
- Tablet and desktop optimized
- Smooth user experience on all devices

⚡ Fast Performance
- Lightweight frontend architecture
- No heavy frameworks required
- Optimized JavaScript implementation
